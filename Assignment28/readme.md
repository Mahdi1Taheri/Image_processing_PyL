## Mosaic blur effect(anonymized face)
this program will apply a mosaic blur effect on face in a video or webcam.<br/>
[![chess-face.jpg](https://i.postimg.cc/BQcpY0P2/chess-face.jpg)](https://postimg.cc/HVnbnF2L)

* To use webcam instead of a video, you can change the path in ```cv2.VideoCapture("\path\...)``` with ```cv2.VideoCapture(0)```
